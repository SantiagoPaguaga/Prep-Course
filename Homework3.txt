Arreglos: a diferencia de las variables, donde solo se podia almacenar un dato, los arreglos funcionan como una coleccion de datos, es decir, se los puede imaginar como un contenedor, donde dentro de ellos podemos guardar muchas cosas, no es necesario que sean del mismo tipo los datos que estarán en la coleccion, pueden convivir perfectamente dentro de un arreglo un numero con una cadena de texto, dentro del arreglo cada elemento se ubicara en una posición determinada, cada posicion puede ser accedida con un indice, asi por ejemplo el primer elemento de nuestro arreglo tendra asignado el indice 0, si queremos acceder a el podemos escribir el nombre del arreglo, luego los corchetes, y dentro de este corchete debemos colocar el indice que tiene el elemento al cual queremos acceder, al igual que los Strings, los arreglos tambien tienen una serie de metodos que podemos utilizar, por ejemplo .length que devuelve la cantidad de elementos del arreglo, .push que es para agregar un elemento al final del arreglo, .pop eliminar el ultimo elemento de nuestro arreglo, .unshift para agregar un elemento en el inicio de nuestro arreglo, es decir en el indice 0, .shift para eliminar el primer elemento de nuestro arreglo.
